# Dogfooding Log — Stage 1 Alpha

> Daily notes from using Apex for real. Capture **every** friction point, bug,
> and idea — however small. "I'll remember it" is how insights get lost.
> Pattern across entries informs Stage 2 phase priorities.

## How to use

- One entry per day you use Apex.
- Severity: **P0** (blocks daily use) · **P1** (painful, workaround exists) ·
  **P2** (annoyance) · **idea** (enhancement).
- P0s get fixed in Phase 13; P1/P2/ideas feed the Stage 2 backlog.
